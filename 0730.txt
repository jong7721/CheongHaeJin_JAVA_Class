인터페이스

클래스가 아니다
인터페이스를 사용하는 클래스는 인터페이스에 정의된 모든 추상 메서드를 반드시 재정의 해야한다.
인터페이스는 상수와 추상 메서드를 가진다.
클래스가 아니기 때문에 상속으로 지정할 수 없다.
인터페이스간의 상속에서는 다중 상속이 허용된다.

interface 인터페이스명 {
    //상수
    타입 상수명 = 값;
    //추상 메소드
    타입 메소드명(매개변수, ...);
    //디폴트 메소드
    derault 타입 메소드명(매개변수, ...) {...}
    //정적메소드
    static 타입 메소드명(매개변수) {...}

}

모든 메소드는 public 으로 선언 다른걸로 생성 불가 (생략가능)
static은 메모리를 고정하는것, 데이터는 변경 가능
